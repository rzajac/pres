Functions

Function declarations


// Function declaration
function square1(x) { return x*x; }

// Function expression
var square2 = function(x) { return x*x; };
var square3 = function square4(x) { return x*x; };

function square5(x) {
    // Return anonymous function
    return function() {
        return x*x;
    }
}

Nested functions


function hypotenuse(a, b) {
    function square(x) { return x*x; }
    return Math.sqrt(square(a) + square(b)); 
}

Objects

Fundamental data types


var point = {
    propName1: 'value1',
    propName2: 'value2'
};

unordered collection of properties

each of which has a name and a value

Object property attributes

  • writable - property value can be set
  • enumerable - property name is enumerable
  • configurable - property can be deleted or its attributes altered

Object attributes

  • prototype - reference to another object from which properties are inherited
  • class - is a string that categorizes the type of an object
  • extensible - flag specifies whether new properties may be added to the object

JavaScript scope

Global variables are defined throughout the program

Local variables are defined throughout the function in which they are declared,

and any nested functions

Function scope and hoisting


function example() {
    var i = 10;
    for (var x = 0; x < i; x++) {
        var tmp = x * 10;
    }
}
                        

function example() {
    var i = 10, x, tmp;
    for (x = 0; x < i; x++) {
        tmp = x * 10;
    }
}
